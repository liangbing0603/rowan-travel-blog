# 🚀 快速开始 - 动态网站部署

## 本地运行（测试）
1. 双击 `start.bat`
2. 访问 http://localhost:3000
3. 测试咨询功能

## 部署到公网（3选1）

### 方法1：Vercel（推荐）
1. 访问 https://vercel.com
2. 用GitHub登录
3. 拖拽本文件夹上传
4. 立即获得网址

### 方法2：Railway
1. 访问 https://railway.app
2. 新建项目
3. 部署，获得网址

### 方法3：Heroku
1. 安装Heroku CLI
2. `heroku create`
3. `git push heroku main`

## 📱 功能测试
1. 提交咨询 → 保存到数据库
2. 查看管理后台 → http://localhost:3000/admin
3. 手机访问 → 响应式设计
