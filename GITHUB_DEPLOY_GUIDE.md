# 🚀 GitHub Pages 部署指南

## 📋 部署前准备
1. **GitHub账号** - 已注册完成
2. **仓库名称** - `rowan-travel-blog`（建议）
3. **网站文件** - 本文件夹中的所有文件

## 🎯 部署步骤（5分钟完成）

### 第1步：创建新仓库
1. 登录GitHub
2. 点击右上角"+" → "New repository"
3. 填写信息：
   - **Repository name:** `rowan-travel-blog`
   - **Description:** `Rowan的个人旅行咨询网站`
   - **选择:** `Public`（必须公开）
   - **勾选:** `Add a README file`
4. 点击"Create repository"

### 第2步：上传网站文件
1. 进入新建的仓库页面
2. 点击"Add file" → "Upload files"
3. **拖拽本文件夹中的所有文件**到GitHub页面
4. 点击"Commit changes"

### 第3步：启用GitHub Pages
1. 点击仓库顶部的"Settings"
2. 左侧菜单点击"Pages"
3. 配置：
   - **Source:** 选择"Deploy from a branch"
   - **Branch:** 选择"main"
   - **Folder:** 选择"/(root)"
4. 点击"Save"

### 第4步：等待部署
1. 等待1-2分钟
2. 刷新页面
3. 看到绿色提示和你的网站网址

## 🌐 你的网站网址
部署成功后，你的网站网址将是：
`https://你的用户名.github.io/rowan-travel-blog`

**示例：** 如果你的GitHub用户名是`rowantravel`，网址为：
`https://rowantravel.github.io/rowan-travel-blog`
