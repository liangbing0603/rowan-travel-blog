## 🚀 部署方法（3选1）

### 方法1：GitHub Pages（推荐）
1. 注册GitHub账号（免费）
2. 创建仓库：`rowan-travel`
3. 上传本文件夹所有文件
4. 设置 → Pages → 选择main分支
5. 获得网址：`https://用户名.github.io/rowan-travel`

### 方法2：Netlify（最简单）
1. 访问 https://netlify.com
2. 拖拽本文件夹上传
3. 立即获得网址

### 方法3：Vercel（快速）
1. 访问 https://vercel.com
2. 用GitHub登录
3. 导入仓库，自动部署
