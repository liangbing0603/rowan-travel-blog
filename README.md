## 🚀 部署步骤

### 第1步：注册GitHub账号
1. 访问 https://github.com
2. 点击"Sign up"
3. 填写信息：
   - 邮箱：你的邮箱
   - 密码：安全密码
   - 用户名：建议用 `rowantravel` 或类似
4. 验证邮箱

### 第2步：创建新仓库
1. 登录GitHub
2. 点击右上角"+" → "New repository"
3. 填写信息：
   - Repository name: `rowan-travel-blog`
   - Description: "Rowan的个人旅行咨询网站"
   - Public（公开）
   - 勾选"Add a README file"
4. 点击"Create repository"

### 第3步：上传文件
1. 进入新建的仓库
2. 点击"Add file" → "Upload files"
3. 拖拽本文件夹中的所有文件
4. 点击"Commit changes"

### 第4步：启用GitHub Pages
1. 进入仓库设置（Settings）
2. 左侧点击"Pages"
3. 配置：
   - Branch: `main`
   - Folder: `/(root)`
4. 点击"Save"

### 第5步：等待部署
1. 等待1-2分钟
2. 刷新页面，看到绿色提示
3. 点击提供的网址测试
